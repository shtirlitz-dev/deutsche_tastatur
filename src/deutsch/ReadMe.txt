========================================================================
WIN32 application.
========================================================================

Deutsche Tastatur is an application that allows you to enter 
additional letters that are not present on the main keyboard layout:

Ä, ä, Ö, ö, Ü, ü, ß - as right Alt + A, a, ...
Right Alt can be pressed together with the main letter (and, if required, Shift)
Or, if it is not supported by the currently used application,
you can first press right Alt, then press (shift+) A.

Additional possibilities:

àá - right Alt + '-', then A, right Alt + '+', then A
èé, ìí - in the same way

"volume-"  - right Alt + [
"volume+"  - right Alt + ]
"prev track" - right Alt + <
"next track" - right Alt + >
"pause, play" - right Alt + /
